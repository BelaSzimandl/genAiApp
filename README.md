# genAiApp

Generative AI demos for stock-market / log insights chatbots.

| Project | Description |
|---------|-------------|
| `stockMarketChat/` | **.NET 10 Blazor WebAssembly** trading-desk chat UI → Foundry agent / SpaceXAI |
| `stockAiApp/` | Python CLI chatbot + Cosmos ingest (specs in `specs/`) |
| `foundryLogAgent/` | Azure AI Foundry agent wiring (`log-insights-chatbot`) |
| `stockQueryApi/` | FastAPI stock quotes API for Foundry tools |
